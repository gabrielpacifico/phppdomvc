<h2> Projeto PHP PDO usando arquitetura MVC e banco de dados PostgreSQL </h2>
<h3> Exemplo de sistema de postagens simples com título e texto </h3>

- Faça a restauração da base de dados que está na raiz da aplicação e insira no seu Pg Admin 4 para conseguir utilizar o banco de dados;
- Crie um usuário na sessão de cadastro, logo apos seus dados serem validados, será redirecionado para o login;
- Após o login será redirecionado à página de postagens onde poderá cadastrar novas postagens, editar e/ou excluir as existentes.
